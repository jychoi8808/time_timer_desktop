## 1. 기본 화면 구성

```
┌─────────────────────────────┐
│ 📌          ⚙  —  ✕        │  ← 상단 바
│                             │
│         [타이머 다이얼]      │  ← 원형 다이얼 (시간 설정 + 표시)
│                             │
│         🔄        ▶️        │  ← 초기화 / 시작·일시정지 버튼
└─────────────────────────────┘
```

| 버튼 | 기능 |
|------|------|
| 📌 | 창을 항상 맨 앞에 고정 / 고정 해제 |
| ⚙ | 설정 화면 열기 |
| — | 창 최소화 |
| ✕ | 앱 종료 |
| 🔄 | 타이머 초기화 |
| ▶️ / ⏸️ | 타이머 시작 / 일시정지 |

---

## 2. 시간 설정

타이머가 **정지 상태**일 때 다이얼을 드래그해 시간을 설정합니다.

- 다이얼 위에 마우스를 올리면 설정될 시간이 미리 표시됩니다.
- **시계 방향**으로 드래그할수록 시간이 늘어납니다.
- 시간은 **1분 단위**로 설정됩니다.
- 기본 최대 시간은 **60분**이며, 설정에서 **120분 모드**로 변경할 수 있습니다.

---

## 3. 타이머 시작 · 일시정지 · 초기화

| 동작 | 방법 |
|------|------|
| **시작** | ▶️ 버튼 클릭 (시간이 설정된 상태에서) |
| **일시정지** | ⏸️ 버튼 클릭 |
| **재개** | ▶️ 버튼 클릭 |
| **초기화** | 🔄 버튼 클릭 → 남은 시간이 0으로 리셋 |

> 타이머가 실행 중일 때는 다이얼 드래그로 시간을 변경할 수 없습니다.

---

## 4. 타이머 종료 알림

시간이 다 되면 알림이 울립니다. 알림은 **클릭하기 전까지 계속 반복**됩니다.

- **소리 알림** (기본): 선택한 알림음이 반복 재생됩니다.
- **무음 모드**: 화면이 빨간색으로 반복해서 깜빡입니다.

**알림 해제 방법**

| 모드 | 해제 방법 |
|------|-----------|
| 일반 모드 | ▶️ 버튼 또는 🔄 버튼 클릭 |
| 미니모드 | 다이얼 중앙 또는 바깥 영역 클릭 |

## 5. 미니모드

창의 크기를 **가로 200px · 세로 250px 이하**로 줄이면 자동으로 미니모드로 전환됩니다.

```
┌──────────┐
│ 📌       │  ← 핀 버튼만 표시
│  ╭────╮  │
│  │MM:SS│  │  ← 시간 표시
│  ╰────╯  │
└──────────┘
```

**미니모드에서 할 수 있는 것**

| 영역 | 동작 |
|------|------|
| 도넛 링 (바깥쪽 원 테두리) | 드래그 → 시간 설정 |
| 중앙 원 / 링 바깥 | 클릭 → 시작 · 일시정지 |
| 📌 버튼 | 항상 위 고정 토글 |
| 창 테두리 · 모서리 | 드래그 → 창 크기 조절 |

> 미니모드로 전환하면 다이얼의 눈금과 버튼이 숨겨지고, 정지 상태에서는 파이 섹터가 회색으로 표시됩니다.

---

## 6. 창 조작

**이동**: 상단 바 영역을 드래그합니다. (미니모드에서는 창 전체 드래그 가능)

**크기 조절**: 창의 테두리나 모서리에 마우스를 올리면 커서가 바뀝니다. 드래그하면 크기를 자유롭게 바꿀 수 있습니다.

- 최소 크기: **100 × 100px**
- 기본 시작 크기: **420 × 620px**

**항상 위 고정**: 📌 버튼으로 토글합니다. 활성화 시 다른 창 위에 항상 표시됩니다. (기본값: 활성화)

## 7. 설정

⚙ 버튼을 눌러 설정 화면을 열 수 있습니다.

### 알림음

| 항목 | 설명 |
|------|------|
| 알림음 사용 | 타이머 종료 시 소리 재생 여부 |
| 무음 모드 | 소리 대신 화면 빨간 깜빡임으로 알림 |
| 알림음 종류 | 7가지 사운드 중 선택 |
| ▶ 미리듣기 | 선택한 알림음 또는 깜빡임 효과 미리 확인 |

**알림음 종류**

| 이름 | 특징 |
|------|------|
| 부드러운 차임 | 은은한 2음 차임 (기본값) |
| 도미솔 상승 | 밝게 올라가는 3음 |
| 종소리 | 은은하고 긴 벨 울림 |
| 디지털 비프 | 짧은 비프음 3회 |
| 딩동 | 현관벨 스타일 2음 |
| 알람 시계 | 짧고 반복적인 알람음 |
| 파도 음 | 부드러운 파도 효과음 |

### 타이머

| 항목 | 설명 |
|------|------|
| 120분 모드 | 다이얼 최대 시간을 60분 → 120분으로 변경 |

### 화면

| 항목 | 설명 |
|------|------|
| 창 투명도 | 슬라이더로 20%~100% 조절 |
