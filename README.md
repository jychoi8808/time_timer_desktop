# Time Timer (Desktop) v2

Google의 Time Timer 제품에서 영감을 받은 반응형 데스크탑 타이머입니다.

## 주요 기능

### ⏱ 시각적 카운트다운
설정한 시간만큼 빨간 면적이 표시되고, 시간이 흐를수록 면적이 부드럽게 줄어들어 남은 시간이 직관적으로 보입니다.

### 📐 반응형 비율 유지
창을 자유롭게 리사이즈할 수 있고, 타이머 다이얼은 항상 1:1 원형 비율을 유지합니다.
눈금, 숫자, 중앙 시간 텍스트, 링 두께가 모두 창 크기에 비례해서 스케일됩니다.
네 모서리와 네 변을 드래그해서 크기를 바꿀 수 있어요.

### 🎨 인터랙티브 디자인
- **호버 미리보기**: 다이얼 위에 마우스를 올리면 연한 빨간색으로 어디까지 설정될지 미리 보여줍니다.
- **상태 표시**: 중앙 시간 아래에 "시간을 설정하세요", "준비", "실행 중", "설정 중..." 같은 상태가 나타납니다.
- **부드러운 애니메이션**: 50ms마다 리페인트되어 빨간 면적이 끊김 없이 줄어듭니다.
- **그라디언트 배경과 그림자**로 입체감 있는 외형.
- **프레임리스 + 둥근 모서리** 모던한 윈도우 스타일.

### 🔔 알림음 8종 (무료 내장)
외부 파일 의존 없이 프로그램 실행 시 파형을 즉석에서 합성해 사용합니다. exe로 배포해도 추가 리소스 파일이 필요 없어요.

- 부드러운 차임 (기본)
- 도미솔 상승
- 종소리
- 디지털 비프
- 딩동
- 알람 시계
- 파도 음
- 무음

드롭다운으로 선택하고 `▶ 미리듣기` 버튼으로 확인할 수 있습니다.

### 📌 Always-on-Top 토글
오른쪽 상단 📌 버튼으로 켜고 끕니다. 기본값 ON.

### 🎯 프리셋 버튼
5 / 10 / 15 / 25 / 45분 원터치 설정 (25분은 뽀모도로용).

## 설치

Python 3.8 이상 필요.

```bash
pip install -r requirements.txt
```

## 실행

```bash
python time_timer.py
```

## Windows exe로 빌드

```bash
pip install pyinstaller
pyinstaller --noconsole --onefile --name "TimeTimer" time_timer.py
```

`dist/TimeTimer.exe`가 생성됩니다. 외부 리소스 파일이 없어서 이 exe 하나만 있으면 동작합니다.

> **참고**: Windows에서 QSoundEffect가 동작하려면 PyQt5의 QtMultimedia 플러그인이 함께 번들링되어야 합니다. PyInstaller가 보통 자동으로 처리하지만, 소리가 안 난다면 빌드 명령에 `--collect-submodules PyQt5` 옵션을 추가해 보세요:
>
> ```bash
> pyinstaller --noconsole --onefile --collect-submodules PyQt5 --name "TimeTimer" time_timer.py
> ```

## 사용 방법

1. **시간 설정**: 다이얼을 클릭하거나 드래그. 12시 방향이 0, 시계방향으로 최대 60분, 5초 단위로 스냅.
2. **시작**: 빨간 `▶ 시작` 버튼.
3. **일시정지 / 리셋**: `⏸ 일시정지` 또는 `⟲ 리셋`.
4. **알림음 선택**: 하단 드롭다운에서 원하는 소리를 고르고 `▶ 미리듣기`로 확인.
5. **창 이동**: 상단 타이틀 영역 드래그.
6. **창 크기 조절**: 모서리/변을 드래그.
7. **항상 위 토글**: 📌 버튼.
8. **최소화 / 종료**: 상단 우측 `—` / `✕` 버튼.

## 커스터마이즈

`time_timer.py`에서:

- **최대 시간 변경**: `TimerDial.max_minutes`를 30, 45, 90 등으로 변경 (다이얼 한 바퀴의 의미가 바뀜).
- **사운드 추가**: `SoundLibrary._generate_all`의 `recipes` 딕셔너리에 새 레시피 추가.
- **색상 변경**: `QColor(239, 68, 68)`, `QColor(185, 28, 28)` 부분이 빨간 그라디언트.
- **프리셋 변경**: `_init_ui` 안의 `[5, 10, 15, 25, 45]`.
- **최소 창 크기**: `self.setMinimumSize(300, 470)`.

## 알려진 제약

- Windows에서 테스트된 구조이지만 macOS/Linux에서도 PyQt5만 깔려 있다면 동작합니다.
- `QSoundEffect`는 WAV 포맷만 지원합니다 (이 프로그램은 WAV만 생성하므로 문제없음).
