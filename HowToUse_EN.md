## 1. Interface Overview

```
┌─────────────────────────────┐
│ 📌          ⚙  —  ✕        │  ← Title bar
│                             │
│         [Timer Dial]        │  ← Circular dial (set & display time)
│                             │
│         🔄        ▶️        │  ← Reset / Start·Pause buttons
└─────────────────────────────┘
```

| Button | Function |
|--------|----------|
| 📌 | Pin window always on top / unpin |
| ⚙ | Open settings |
| — | Minimize window |
| ✕ | Quit app |
| 🔄 | Reset timer |
| ▶️ / ⏸️ | Start / Pause timer |

## 2. Setting the Time

Drag the dial to set the duration while the timer is **stopped**.

- Hovering over the dial previews the time that would be set.
- Drag **clockwise** to increase the time.
- Time is set in **1-minute increments**.
- The default maximum is **60 minutes**. Switch to **120-minute mode** in Settings if needed.

---

## 3. Start · Pause · Reset

| Action | How |
|--------|-----|
| **Start** | Click ▶️ (with time already set) |
| **Pause** | Click ⏸️ |
| **Resume** | Click ▶️ |
| **Reset** | Click 🔄 — remaining time returns to 0 |

> You cannot adjust the dial while the timer is running.

---

## 4. End-of-Timer Alert

When time runs out, an alert fires and **keeps repeating until you click to dismiss it**.

- **Sound alert** (default): the selected sound plays on a loop.
- **Silent mode**: the window flashes red repeatedly instead.

**How to dismiss the alert**

| Mode | Dismiss by |
|------|-----------|
| Normal mode | Clicking ▶️ or 🔄 |
| Mini mode | Clicking the dial center or the area outside the ring |

## 5. Mini Mode

Resize the window to **under 200 px wide or 250 px tall** and it switches to Mini Mode automatically.

```
┌──────────┐
│ 📌       │  ← Pin button only
│  ╭────╮  │
│  │MM:SS│  │  ← Time display
│  ╰────╯  │
└──────────┘
```

**What you can do in Mini Mode**

| Area | Action |
|------|--------|
| Donut ring (outer band) | Drag → set time |
| Center circle / outside ring | Click → start · pause |
| 📌 button | Toggle always-on-top |
| Window edge / corner | Drag → resize window |

> In Mini Mode, tick marks and control buttons are hidden. The pie sector turns grey when the timer is paused.

---

## 6. Window Controls

**Move**: Drag the title bar area. (In Mini Mode, drag anywhere on the window.)

**Resize**: Hover over any edge or corner — the cursor changes to a resize arrow. Drag to resize freely.

- Minimum size: **100 × 100 px**
- Default size: **420 × 620 px**

**Always on Top**: Toggle with the 📌 button. When active, the window stays above all other windows. (Default: on)

## 7. Settings

Click ⚙ to open the Settings panel.

### Alert Sound

| Option | Description |
|--------|-------------|
| Enable alert sound | Play a sound when the timer ends |
| Silent mode | Flash the screen red instead of playing a sound |
| Sound type | Choose from 7 built-in sounds |
| ▶ Preview | Try the selected sound or flash effect |

**Available sounds**

| Name | Character |
|------|-----------|
| Soft Chime | Gentle two-note chime (default) |
| Do-Mi-Sol Rise | Bright ascending three-note tone |
| Bell | Warm, sustained bell ring |
| Digital Beep | Three short electronic beeps |
| Ding-Dong | Doorbell-style two-note tone |
| Alarm Clock | Short, rapid repeating alarm |
| Wave | Soft ocean wave effect |

### Timer

| Option | Description |
|--------|-------------|
| 120-minute mode | Extend the dial maximum from 60 to 120 minutes |

### Display

| Option | Description |
|--------|-------------|
| Window opacity | Adjust transparency from 20% to 100% using the slider |

| Light mode | Dark mode |
|------------|-----------|
| White background, bright dial | Dark background, deep-toned dial |
